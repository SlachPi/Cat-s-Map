Welcome to Cat_Editor, a basic browser based editor. Cat_Editor enables you to create nice html pages easily.

## Basic Usage :#

The above is the way you do a title.

**And this is a bold text**

Initialize a list like this :

#LIST#

Initialize an enumeration list :

#ENUM#

And this is an inline equation : $E=mc^2$ (a famous one). If you don't want it inline double the dollar signs.

This is how you initialize (say) a 4 by 3 table :

#4#3#

You can also put images :

#IMG#https://catsmap.tk/cat.svg#

And Links :

#LINK#https://www.google.com/#This is a link to google.com#

You can also initialize a code bloc like that :

#CODE#

Click on the "compile" button above to see the results and the "save" button to save the result as an html file.
